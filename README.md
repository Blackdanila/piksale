# PIKsale

Мониторинг и аналитика цен на квартиры застройщика ПИК.

**https://piksale.ru**

## Возможности

- Ежедневный сбор данных с API ПИК (квартиры, цены, планировки)
- Отслеживание базовой цены и цены со скидкой (benefitPrice)
- Фиксация снятых с продажи квартир (статус `gone`)
- История изменения цен с графиками динамики
- Топ снижений цен и топ ЖК по продажам
- Telegram-бот с уведомлениями об изменении цен и подписками на ЖК
- Дневная агрегированная статистика по каждому ЖК

## Стек

- **Runtime:** Node.js 22, TypeScript
- **Web:** Hono (SSR, HTML-шаблоны)
- **DB:** PostgreSQL 16 + Prisma ORM
- **Bot:** grammY (Telegram, webhook)
- **Деплой:** Docker Compose

## Структура

```
src/
  index.ts          — точка входа, запуск сервера и бота
  scheduler.ts      — cron-задача сбора данных (08:00 MSK)
  scraper/          — клиент API ПИК, коллектор, агрегатор
  web/              — веб-приложение (страницы, компоненты, API)
  bot/              — Telegram-бот (команды, хэндлеры)
  db/               — Prisma-клиент, кешированные запросы
  formatters/       — форматирование карточек квартир для бота
prisma/
  schema.prisma     — схема БД
```

## Запуск

```bash
cp .env.example .env  # BOT_TOKEN, WEBHOOK_URL (опционально)
docker compose up -d
```

Приложение запустится на `http://localhost:3100`. При старте контейнера автоматически применяются миграции БД.

### Локальная разработка

```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
```

## Переменные окружения

| Переменная     | Описание                          | Обязательная |
|----------------|-----------------------------------|:---:|
| `BOT_TOKEN`    | Токен Telegram-бота               | да  |
| `DATABASE_URL` | PostgreSQL connection string      | да  |
| `WEBHOOK_URL`  | URL для вебхука бота (если есть)  | нет |
| `PORT`         | Порт сервера (по умолчанию 3000)  | нет |

## Сбор данных

Данные собираются автоматически каждый день в 08:00 MSK. Принудительный запуск:

```bash
docker exec piksale-app node dist/scraper/collector.js
```

## Бэкапы

БД бэкапится ежедневно в 01:00 UTC в `/opt/piksale/backups/`. Хранятся 30 дней.
